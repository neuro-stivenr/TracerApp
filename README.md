# TracerApp

Simple app to see whether a patient is clear for PET.

## Installation

```bash
git clone https://github.com/umich-stivenr/TracerApp
cd TracerApp
npm install
```

## Usage

```bash
npm start
```

Then, open your browser to:
http://localhost:5656

Alternatively, try out the live version of this app at:
https://tracerapp.xyz
